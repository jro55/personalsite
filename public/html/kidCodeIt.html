1: Set up a blank HTML document that links to a CSS stylesheet.

The main goal of this problem is to successfully set up an HTML page that links to a CSS stylesheet.

  • Create a new folder on your desktop called "Kid Code It" to save all new files into.
  • Open a blank Sublime document and save it to your new Kid Code It folder as main.html.
  • Properly set up the HTML document using <!DOCTYPE>, <html></html>, <head></head>, and <body></body> tags.
  • Create a <h1></h1> tag that displays the text "Will's Journey to Becoming a Programming Wiz".
  • Open another blank sublime document and save it as main.css.
  • In your html document, link to your new main.css stylesheet by inserting a <link> tag inside of the <head>.
    The link tag should look like this <link rel="stylesheet" type="text/css" href="main.css">.
  • Using classes, change the background-color of your web page as well as the color of your <h1> header.
