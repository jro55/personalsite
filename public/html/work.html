Completed and Ongoing Projects

StandUp!

Stand up is a site built for stand up comedians as well as anyone who enjoys stand up comedy. An original idea used as my final project for RefactorU to showcase my skills. StandUp utilizes the MEAN Stack for the backbone of site, Twitter Bootstrap for the grid system, TokBox API for live video streaming, and websockets to make everything real time.

While this project was fun and I would love to fully complete it and make it live, other work has forced me to postpone this project until further notice.

View the gitHub repository here.

The ZONE

The ZONE project was part of the GiveFirst Hackfest put on by TechStars. My team, consisting of two other people and myself, volunteered to take on the project of creating an entire online presence for a non-profit located in Denver. This site included a e-commerce store, blog, and donation page. We had a total of 20 hours to complete this project. My direct part in this was designing and building the home page as well as helping out others.

View the website at www.thezonedenver.org.

Dev Camp Grads

Dev Camp Grads is a small for-practice project I choose to do to familiarize myself with the Angular Material framework. This project introduced me to all the beatiful and fluid animations that Google Material has created. These beutiful animations created by Google paired with Angular has made me fall in love with Angular Material. Not only does it make great interactive sites for desktop, but it also excels on the mobile level as well.

Solar System

Solar System was a group project I did with several friends just to learn a new JavaScript library and further our knowledge. We utilized the Three.js library to create an interactive solar system. This project gave us a basic understanding of the library including how to utilize scenes and cameras in three.js.
